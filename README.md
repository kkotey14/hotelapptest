# üè® Hotel Reservation System

A full-stack web application designed to streamline hotel operations. This system features a multi-role authentication structure, allowing Guests to book rooms, Staff to manage reservations, and Administrators to oversee the entire hotel inventory and user base.

## üöÄ Key Features

### Guest User
* **Secure Registration & Login:** Create personal accounts to track booking history.
* **Room Browsing:** View available rooms with details on amenities and pricing.
* **Booking Engine:** Select dates and book rooms in real-time.
* **Booking Status:** View current reservation status (Pending/Confirmed).

###  Staff User
* **Reservation Management:** View incoming booking requests.
* **Check-in/Check-out:** Update room status as guests arrive or leave.
* **Guest Assistance:** Access guest details for quicker service at the front desk.

###  Admin User
* **Dashboard Overview:** Visual metrics on occupancy and total users.
* **Room Inventory:** Add, edit, or delete rooms and set pricing.
* **User Management:** Manage staff and guest accounts (approve/ban users).
* **System Settings:** Configure global site settings.
